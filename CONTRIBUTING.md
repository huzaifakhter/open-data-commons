# Contributing to Open Data Commons

Thank you for your interest in contributing! We welcome datasets from all domains as long as they meet our quality and legal standards.

## Contribution Rules

1. **Public Domain Dedication**: By contributing to this repository, you agree to release your contribution under the **CC0 1.0 Universal** license. All data here must be free of copyright restrictions.
2. **Legal Compliance**: Only share data that you have the legal right to share. Do not include copyrighted, private, or sensitive individual data (PII).
3. **Format**: We prefer data in **CSV** format for its simplicity and broad compatibility.
4. **Documentation**: Each new dataset folder must include a `README.md` explaining:
   - What the data represents.
   - Column definitions.
   - Data source or collection methodology.
   - Any processing or cleaning steps performed.
5. **Clean Data**: Ensure formatting is consistent (lowercase headers preferred), no duplicates, and no unnecessary metadata files.

## How to Contribute

1. Fork the repository.
2. Create a new directory under `datasets/` named after your domain.
3. Add your data file and a `README.md`.
4. Submit a Pull Request.

Together, we can build a truly open data ecosystem.
